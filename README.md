## Time Series Analysis Project

По теме: "Для каких типов рядов какая трансформация ряда полезнее?"

Автор: Гусев Владислав, ИСИН25

Состав репозитория:

1. config.py - конфигурация запусков моделей
2. run_experiment.py - файл для запуска всех экспериментов
3. src - папка с основным кодом проекта

  3.1. baseline_model.py - расчет метрик бейзлайн моделей

  3.2. global_model.py - расчет метрик рекурсивной CatBoost модели

  3.3. metrics.py - расчет метрик и построение графиков для оценки качества модели

  3.4. preprocessing.py - предобработка сырого датасета

  3.5. transformations.py - прямые и обратные преобразования рядов

  3.6. utils.py - полезные функции для преобразования датасетов для обучения

4. results - папка с ноутбуком, в котором запускались эксперименты и визуализация
   4.1. analysis_results.ipynb - ноутбук для запуска

5. project_report.pdf - исследовательский отчет о проделанных экспериментах
