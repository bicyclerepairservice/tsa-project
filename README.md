## Time Series Analysis Project

Состав репозитория:

1. config.py - конфигурация запусков моделей
2. src - папка с основным кодом проекта

  2.1. baseline_model.py - расчет метрик бейзлайн моделей

  2.2. global_model.py - расчет метрик рекурсивной CatBoost модели

  2.3. metrics.py - расчет метрик и построение графиков для оценки качества модели


  2.4. preprocessing.py - предобработка сырого датасета

  2.5. transformations.py - прямые и обратные преобразования рядов

  2.6. utils.py - полезные функции для преобразования датасетов для обучения

3. results - папка с ноутбуком, в котором запускались эксперименты и визуализация

   3.1. analysis_results.ipynb - ноутбук для запуска
